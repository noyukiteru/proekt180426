Прототип системы записи на шиномонтаж

Автоматизированная информационная система для записи клиентов на услуги шиномонтажа. Включает Telegram-бота для клиентов, веб-дашбор* для администраторов и централизованный сервер на FastAPI с реляционной базой данных.


Система построена с разделением на три модуля, взаимодействующих через REST API:
- Telegram-бот (клиентский модуль)
  Пошаговая запись через FSM, каталог услуг, выбор мастера, генерация свободных слотов, управление записями, сбор отзывов.
- Веб-дашборд (административно-аналитический модуль)
  Авторизация персонала, CRUD для услуг/мастеров/записей, фильтрация расписания, визуализация аналитики (Chart.js).
- REST API (ядерный модуль)
  19 CRUD-эндпоинтов, асинхронная работа с БД, валидация через Pydantic, Dependency Injection для сессий БД.

Установка и запуск
1. Установите Python 3.10+ и создайте виртуальное окружение:
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt

2. Создайте файл .env в корне проекта и укажите токен бота (получить у @BotFather):
BOT_TOKEN=ваш_токен
ADMIN_LOGIN=admin
ADMIN_PASSWORD=admin123
DATABASE_URL=sqlite:///./app.db
SECRET_KEY=supersecretkey123

3. Заполните базу тестовыми данными (один раз):
python seed_db.py

4. Запустите сервер (в первом терминале):
uvicorn dashboard.main:app --reload

5. Запустите бота (во втором терминале):
python -m bot.main
